## üß† 1. The Memory Implant: Custom Instructions & Copilot Memory

Here's what most people miss about Copilot: **It forgets everything between sessions.** Every conversation starts from zero. Every email draft ignores the 40 emails you wrote yesterday with the same signature, same tone, same structure.

Until now.

### Custom Instructions: Your Global Preferences

Think of Custom Instructions as the **system prompt for your entire work life**. You tell Copilot once‚Äî"I'm a product manager at a SaaS startup, I prefer concise emails, always include action items"‚Äîand it applies that context to *every single interaction*.

**How to Set It Up:**

1. Open Microsoft 365 Copilot (copilot.microsoft.com or the app)
2. Navigate to **Settings** (gear icon, top right)
3. Select **Personalization** ‚Üí **Custom Instructions**
4. Write your instructions in two sections:
   - **What would you like Copilot to know about you?** (Your role, company, preferences)
   - **How would you like Copilot to respond?** (Tone, format, constraints)

<MDXImage src="images/custom-instructions-menu.jpg" alt="Copilot personalization menu showing Custom Instructions and Copilot Memory" caption="Settings ‚Üí Personalization. If you see a disabled panel, your admin may have turned personalization off." />

**Example Custom Instructions:**

```text
ABOUT ME:
- Role: Engineering Manager at a fintech company
- Team: 8 developers, distributed across 3 time zones
- Primary tools: Jira, Slack, GitHub, Excel
- Communication style: Direct, data-driven, avoid corporate jargon

HOW TO RESPOND:
- Keep emails under 5 sentences unless I ask for detailed analysis
- Always structure responses with clear headings and bullet points
- When drafting messages, include a TL;DR at the top
- For data analysis, lead with the insight, then show the numbers
- Never use phrases like "I hope this email finds you well"
```

<ConceptCard title="Why This Matters" icon="üí°">

**Custom Instructions = Context Compression.**

Without them, every prompt needs 50 extra words of context. "I'm an engineering manager, my team uses Jira, I need a concise update for my VP..."

With them? You just say: **"Draft the weekly update."**

Copilot already knows the rest.

</ConceptCard>

### Copilot Memory: The Living Context

This is the game-changer. **Copilot Memory** is a persistent knowledge base that Copilot references across *all* your conversations and apps.

**Key Differences:**

| Feature | Custom Instructions | Copilot Memory |
|---------|---------------------|----------------|
| **Scope** | Global preferences (tone, role, format) | Specific facts, project details, relationships |
| **Lifespan** | Permanent until you change it | Evolves as you tell it to remember things |
| **Example** | "I prefer bullet points" | "Project Phoenix launches March 15th" |

**How to Use Copilot Memory:**

1. During any conversation, say: **"Remember that Project Phoenix launches on March 15th, 2026."**
2. Copilot stores this fact in your memory bank.
3. Next week, you ask: **"What's the status of our upcoming launches?"**‚ÄîCopilot recalls Phoenix automatically.

**Viewing & Managing Memory:**

- Go to **Settings** ‚Üí **Personalization** ‚Üí **Copilot Memory**
- See a list of all stored facts
- Delete outdated ones (e.g., when a project ends)

## üìä 4. Excel Upgrades: Agent Mode & =COPILOT() Function

This is where Copilot stops being a chatbot and becomes a **data analyst sitting in your spreadsheet**.

### Agent Mode (Excel Labs Add-In)

**What It Does:**

Normal Copilot in Excel is reactive‚Äîyou ask, it responds with text. **Agent Mode** is *proactive*‚Äîyou give it a goal, and it creates tables, charts, and pivot analyses *directly in your workbook*.

**Setup:**

1. Open Excel (web version: office.com/launch/excel)
2. Go to **Insert** ‚Üí **Get Add-ins**
3. Search for **"Excel Labs"**
4. Install the add-in (free, built by Microsoft Research)
5. Open the Excel Labs pane (appears on the right side)
6. Enable **"Copilot Agent Mode"** toggle

<MDXImage src="images/excel-copilot-tutorial.jpg" alt="Copilot in Excel showing insights and a chat panel" caption="Copilot inside Excel turns natural language into tables, charts, and insights." />

<Callout type="warning">
**Availability:** Excel Labs and Agent Mode are experimental and may be tenant/region dependent. If you don't see them, use Copilot in Excel Chat (right sidebar) for similar analysis workflows.
</Callout>

<Callout type="info">
**Fallback if `=COPILOT()` is missing:** Use the Copilot chat pane to label sentiment, then paste results into column C. The lab still works‚Äîyou just apply the results manually.
</Callout>

**Example Mission:**

You have a sales data table with columns: Date, Product, Region, Revenue, Units Sold.

Instead of writing formulas, you say:

```text
Create a pivot table showing total revenue by Region and Product,
sorted by highest revenue. Then create a bar chart comparing regions.
```

Copilot Agent Mode:
1. Inserts a new sheet
2. Builds the pivot table
3. Generates the chart
4. Returns to the original sheet and highlights what it created

**The game changed.** You didn't touch a single formula.

### The =COPILOT() Function

This is **GPT inside a cell**.

**Syntax:**

```excel
=COPILOT("Your prompt here", [optional reference cells])
```

**Real-World Examples:**

| Goal | Formula |
|------|---------|
| Categorize text | `=COPILOT("Categorize this email subject as Urgent/Normal/Low: " & A2)` |
| Extract insights | `=COPILOT("What emotion does this customer review express?", B5)` |
| Reference data | `=COPILOT("Is the revenue in cell C2 above or below our Q4 target in D10?", C2, D10)` |

<ConceptCard title="Why This Is Revolutionary" icon="üî•">

**Before:** You needed nested IF statements, VLOOKUP chains, or VBA macros to process text.

**Now:** `=COPILOT("Extract the company name from this email signature", A2)` ‚Üí Done.

You're writing **natural language formulas**. No syntax memorization. No Stack Overflow searches.

</ConceptCard>

**Natural Language Queries in Chat:**

Instead of using formulas, you can also just *ask* Copilot directly in the Excel chat pane:

- "What was the lowest revenue month in 2025?"
- "Which product had the steepest sales decline quarter-over-quarter?"
- "Highlight cells where Units Sold is above 100 and Revenue is below $5000."

Copilot processes the question, identifies the relevant cells, and gives you the answer with references.

## üìß 5. Outlook Superpowers: AI Prioritization & Draft Instructions

Your inbox is chaos. Not because you're disorganized‚Äîbecause **everyone else's urgency becomes your priority**.

Copilot in Outlook has three nuclear features most people never configure:

### Feature 1: AI Prioritization Rules

Instead of manually flagging emails, you teach Copilot **natural language rules** for what matters.

**How to Set It Up:**

1. Open Outlook (web or desktop)
2. Go to **Settings** ‚Üí **Mail** ‚Üí **Rules** ‚Üí **Copilot Priority Rules**
3. Create rules like:

```text
Mark as HIGH priority:
- Emails from my manager or VP
- Any email with "URGENT" or "ASAP" in subject from internal senders
- Calendar invites from C-level executives

Mark as LOW priority:
- Newsletters or promotional emails
- Automated notifications from Jira/GitHub unless they mention my name
- CC-only emails where I'm not directly addressed
```

4. Copilot applies these rules in real-time
5. Your inbox now has **High/Medium/Low** visual tags

**Result:** You open Outlook, immediately see the 5 emails that *actually* need your attention, and batch-process the rest later.

<Diagram type="dashboard-ui" />

### Feature 2: Thread Summarization

Ever open a 47-email thread and have no idea what's happening?

**Solution:**

1. Open any email thread
2. Look for the **"Summary by Copilot"** button at the top
3. Click it
4. Copilot reads the entire thread and gives you:
   - **Key decisions made**
   - **Outstanding questions**
   - **Action items assigned to you**

**Example Output:**

> **Thread Summary:**
> - **Decision:** Launch delayed from March 1 ‚Üí March 15
> - **Blocker:** Legal review still pending on Terms of Service
> - **Action for you:** Send updated timeline to partners by EOD Friday

### Feature 3: Draft Instructions (The Secret Weapon)

This is like Custom Instructions, but **only for email drafting**.

**How to Configure:**

1. Outlook **Settings** ‚Üí **Copilot** ‚Üí **Draft Instructions**
2. Define your email writing style:

<MDXImage src="images/outlook-draft-instructions-settings.png" alt="Outlook settings showing Copilot Draft Instructions" caption="In new Outlook: Settings ‚Üí Copilot ‚Üí Draft Instructions." />

```text
TONE:
- Professional but conversational
- Avoid corporate buzzwords ("circle back", "touch base", "synergy")

LENGTH:
- Keep emails under 4 sentences unless I specify otherwise
- Always include a clear ask or next step

STRUCTURE:
- Start with context (1 sentence)
- State the ask (1 sentence)
- Provide options or deadline (1 sentence)
- Sign off with first name only, no title

NEVER:
- Use "I hope this email finds you well"
- End with "Best regards" (use "Thanks," instead)
- Write paragraphs longer than 3 lines
```

3. Save Draft Instructions

<MDXImage src="images/outlook-draft-copilot.jpg" alt="Draft with Copilot prompt panel in Outlook" caption="Outlook‚Äôs Copilot panel offers suggested prompts and quick draft options." />

**Usage:**

Copilot writes:

**After Draft Instructions:**

Same prompt. Copilot writes:
> "Sarah‚ÄîQuick check: when will the design mockups be ready? We need them by Friday to stay on schedule for the March 15 launch. Let me know if that's doable. Thanks, [First Name]"

**Same request. Completely different output.** Because Copilot learned your style.
