## 2. Firebase MCP Integrace ğŸ”¥

Skutecna sila Antigravity se odemkne s **Firebase MCP**. Tato integrace dava AI agentovi plnou kontrolu nad vasim backendem.

### Co Firebase MCP umoznuje

| Schopnost | Co AI muze delat |
| --------- | ---------------- |
| **Autentizace** | Nastavit email/heslo, Google OAuth, magic links |
| **Firestore** | Vytvaret kolekce, definovat schemata, psat pravidla |
| **Hosting** | Deployovat do produkce jednim klikem |
| **Storage** | Konfigurovat upload souboru a pristupova pravidla |
| **Functions** | Vytvorit serverless backend logiku |

### Nastaveni Firebase MCP

1. Otevrte okno agenta v Antigravity
2. Kliknete na "MCP Servers" v toolbaru
3. Vyberte "Firebase" ze seznamu
4. Kliknete "Install"

<Callout type="warning">
**Bezpecnostni poznamka:** Firebase MCP ma plny pristup k backendu. Pouzivejte pouze pro projekty kde duverujete akcim AI. Pro produkcni aplikace vzdy zkontrolujte vygenerovana bezpecnostni pravidla.
</Callout>

```text
Vytvor produktivitni aplikaci s:
- Prihlaseni pres email a heslo
- Cloud Firestore databaze pro uzivatele tasky
- Real-time synchronizace mezi zarizeni
- Dark mode UI s tailwind styly
- Deploy na Firebase hosting
```

AI udela:
1. Inicializuje Firebase projekt
2. Nakonfiguruje autentizaci
3. Vytvori Firestore kolekce se spravnymi pravidly
