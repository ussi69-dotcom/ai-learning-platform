## 2. Firebase MCP Integration ğŸ”¥

The real power of Antigravity unlocks with **Firebase MCP**. This integration gives the AI agent full control over your backend.

### What Firebase MCP Enables

| Capability | What AI Can Do |
| ---------- | -------------- |
| **Authentication** | Set up email/password, Google OAuth, magic links |
| **Firestore** | Create collections, define schemas, write rules |
| **Hosting** | Deploy to production with one click |
| **Storage** | Configure file uploads and access rules |
| **Functions** | Create serverless backend logic |

### Setting Up Firebase MCP

1. Open the agent window in Antigravity
2. Click "MCP Servers" in the toolbar
3. Select "Firebase" from the available list
4. Click "Install"

<Callout type="warning">
**Security Note:** Firebase MCP has full backend access. Only use it for projects where you trust the AI's actions. For production apps, always review the generated security rules.
</Callout>

### Example: Full-Stack App in One Prompt

```text

- Email and password signin

- Deploy to Firebase hosting
```

The AI will:
1. Initialize Firebase project
2. Configure authentication
3. Create Firestore collections with proper rules
4. Generate the complete frontend

**Time from prompt to deployed app: ~10 minutes.**
