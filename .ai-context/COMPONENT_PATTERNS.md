## 1. Difficulty Badge
**Location:** Fixed Top-Right
**Visuals:**
- Gradient background (Beginner: green->blue, etc.)
- Border: 2px white
- Shadow: xl
- Hover: scale-105

## 2. Jedi/Sith Toggle
**Location:** Fixed Top-Left
**Visuals:**
- Glass panel container
- Active: colored, scale-105
- Inactive: gray-200
- Border: 2px (gray-300 / red-600)

## 3. ConceptCard
**Layout:** Grid (1-3 cols)
**Visuals:**
- Solid background (white / slate-800)
- Border: 2px colored
- Shadow: lg -> 2xl
- No backdrop-blur!

## 4. Lab Section
**Visuals:**
- Solid background
- Border: 2px purple/red
- Numbered steps with badges
- Completion button (gradient)

## 5. XP Progress Bar
**Location:** Fixed Bottom
**Visuals:**
- Clippy Avatar (Left)
- Progress Bar (Center, gradient fill)
- Next Stage Avatar (Right, opacity-50)
- Border: 2px purple/red
